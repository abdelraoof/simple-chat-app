# Scopic PHP/Laravel Test Task: Simple chat application

## How to use
- Install dependencies
```
$ composer install
$ npm install
```
- Edit env
```
$ cp .env.example .env
$ php artisan key:gen
```
- Run PHP server & Socket.IO server
```
$ php artisan serve
$ npm run start-socket-server
```
- Then point your browser to http://localhost:8000
